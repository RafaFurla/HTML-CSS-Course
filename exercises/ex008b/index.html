Like Jeff Noble said in his book: HTML para Leigos

> Os arquivos HTML que produzem as páginas web são simplesmente documentos de texto, assim como os arquivos XHTML e CSS. Esse uso de documentos de texto é a razão para a web funcionar tão bem quanto funciona. O texto é uma linguagem universal dos computadores. Qualquer arquivo de texto que você crie em um computador com o Windows - incluindo qualquer arquivo HTML, XHTML ou CSS - funciona igualmente bem em um Mac ou em qualquer outro sistema operacional. Porém, as páginas web não são meramente documento de texto. Elas são feitas com um texto sem atenção especial à beleza chamado HTML, XHTML ou CSS. Cada um deles usa seu prórpio conjunto específico de instruções que você inclui (junto com seu conteúdo) dentro de arquivos de texto para especificar como uma página deve ficar e se comportar.
